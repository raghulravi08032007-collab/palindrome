# Palindrome Checker App

## 📌 Project Description
The Palindrome Checker App is a simple Java console application that checks whether a given word or number is a palindrome.

A palindrome is a word, number, or phrase that reads the same forward and backward.

Examples:
- madam → Palindrome
- 121 → Palindrome
- hello → Not a palindrome

## 🚀 Features
- Accepts user input from keyboard
- Checks whether the input is a palindrome
- Displays appropriate result message
- Simple and easy-to-understand Java implementation

## 🛠️ Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

## ▶️ How to Run the Program

Compile the program:
javac PalindromeCheckerApp.java

Run the program:
java PalindromeCheckerApp
